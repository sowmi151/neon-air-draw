# ✨ Neon Air Draw

Neon Air Draw is a browser-based hand tracking drawing application that lets users draw in the air using only their index finger. It uses **MediaPipe Hands** for real-time hand gesture recognition and HTML5 Canvas to create smooth neon glowing strokes.

## 🚀 Features

- ✋ Real-time hand tracking using MediaPipe Hands
- ☝️ Draw with your index finger
- 🌈 Animated neon color-changing drawing effect
- 🤏 Pinch gesture to pause or resume drawing
- ✊ Fist gesture to instantly clear the canvas
- 📱 Works directly in a web browser
- 🎨 Smooth glowing brush effect
- 📷 Live camera background

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- HTML5 Canvas API
- MediaPipe Hands
- MediaPipe Camera Utils

## 🎮 Controls

| Gesture | Action |
|---------|--------|
| ☝️ Index Finger Up | Draw |
| 🤏 Pinch | Pause / Resume Drawing |
| ✊ Fist | Clear Canvas |

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run it using a local web server (recommended).
4. Allow camera permission when prompted.
5. Click **Start Drawing**.
6. Use your hand gestures to draw in the air.
